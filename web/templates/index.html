About

The idea for TeamZone was inspired by a passion for sports and a desire to make it easier for local teams to connect and organize games. We recognized a need for a streamlined platform that would allow teams to quickly and easily find and schedule games with other local teams. With this goal in mind, we set out to develop an application that would simplify the process of organizing games and promote greater community involvement in local sports.

This came to life as a portfolio project for the foundations section of the ALX Software engineering program.

Front-End

This app's UI is created using raw HTML & CSS with some Bootstrap built-in components. The interactivity is based on JavaScirpt (JQuery).

Back-End

We used Flask (Python) as a Back-End framework, storing our data in MySQL DBMS using the sqlalchemy ORM.

Architecture

TeamZone's user interface is populated with data server-side using Jinja templates. The HTML pages are returned ready to be rendered on the client-side, with the exception of the search and filter page, which is populated using JavaScript that fetches data from the application's API. This approach provides a seamless and responsive user experience, with fast page load times and up-to-date information on all teams and games.
