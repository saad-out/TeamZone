About

This web application `TeamZone` is a portfolio project for the foundations section of the ALX Software engineering program.

Starting from the ground up with no programming background required, this project is the result of 9 months of learning. Beginning with the very basics of programming, then advancing to cover every aspect of the web including the Front-End, Back-End and DevOps parts of an application.

read more

Front-End

This app's UI is created using raw HTML & CSS with some Bootstrap built-in components. The interactivity is based on JavaScirpt (JQuery).

Back-End

We used Flask (Python) as a Back-End framework, storing our data in MySQL DBMS using the sqlalchemy ORM.

DevOps

Our infrastructure consists of two Nginx web servers sitting behind an HAProxy load balancer. Forwarding requests to our JavaScirpt code which communicates with the Back-End using RESTful APIs.
